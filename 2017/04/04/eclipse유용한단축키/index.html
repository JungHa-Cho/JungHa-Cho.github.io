Eclipse 유용한 단축키

실행

  1. Ctrl + F11 : 바로 전에 실행했던 클래스 실행 소스 네비게이션

  2. Ctrl + 마우스커서(혹은 F3) : 클래스나 메소드 혹은 멤버를 상세하게 검색하고자 할때

  3. Alt + ->, Alt + <- : 이후, 이전

  4. Ctrl + o : 해당 소스의 메소드 리스트를 확인하려 할때

  5. F4 : 클래스명을 선택하고 누르면 해당 클래스의 Hierarchy 를 볼 수 있다.

  6. Ctrl + Shift + R : 파일 쉽게 열기 (파일명으로 검색) 문자열 찾기

  7. Ctrl + K : 찾고자 하는 문자열을 블럭으로 설정한 후 키를 누른다.

  8. Ctrl + Shift + K : 역으로 찾고자 하는 문자열을 찾아감.

  9. Ctrl + J : 입력하면서 찾을 수 있음.

  10. Ctrl + Shift + J : 입력하면서 거꾸로 찾아갈 수 있음.

  11. Ctrl + F : 기본적으로 찾기

소스 편집

  1. Ctrl + Shift + F : 코드 자동 정리
  2. Ctrl + Shift + G : 특정 메써드나 필드를 Reference하고 있는 곳을 찾는다.
  3. Ctrl + 1 : Quick Fix. 에러가 발생했을 경우 Quick Fix를 통해 쉽게 해결이 가능하다.
  4. CTRL + L : 특정 줄번호로 가기
  5. Alt + Shift + J : 자동으로 주석 달기 (메소드나 멤버변수에 포커스 두고 실행)
  6. F2 : 컴파일 에러의 빨간줄에 커서를 갖져다가 이 키를 누르면 에러의 원인에 대한 힌트를 제공한다.
  7. Ctrl + L : 원하는 소스 라인으로 이동
    로컬 히스토리 기능을 이용하면 이전에 편집했던 내용으로 변환이 가능하다.
  8. Ctrl + Shift + Space : 메소드의 가로안에 커서를 놓고 이 키를 누르면 파라미터 타입 힌트를 볼 수 있다.
  9. 한 줄 삭제 CTRL + D
  10. 파일 닫기 : CTRL+W
  11. 들여쓰기 자동 수정. (3.0 NEW) : CTRL+I
  12. 블록 주석(/…/) 추가.(3.0 NEW): CTRL+SHIFT+/
  13. Ctrl + / 해주면 여러줄이 한꺼번에 주석처리됨. 주석 해제하려면 반대로 하면 됨.
  14. 위(아래)줄과 바꾸기 : ALT+UP(DOWN)
  15. 블록 선택하기. : ALT+SHIFT+방향키
  16. 메소드의 파라메터 목록 보기. : CTRL+SHIFT+SPACE
  17. 자동으로 import 하기 : CTRL+SHIFT+O
  18. 열린 파일 모두 닫기 : CTRL + SHIFT + F4
  19. 전체화면 토글 : CTRL+M
  20. 한 줄(블럭) 복사 : Ctrl + Alt + 위(아래)
  21. 다음 annotation(에러, 워닝, 북마크 가능)으로 점프 : Ctrl + , or .
  22. 메소드 정의부로 이동 : F3
  23. 하이어라키 팦업 창 띄우기(인터페이스 구현 클래스간 이동시 편리) : Ctrl + T
  24. 메소드나 필드 이동하기 CTRL + O
  25. ULTRAEDIT나 EDITPLUS 의 CTRL+TAB 과 같은 기능. : CTRL+F6

템플릿 사용

  1. sysout 입력한 후 Ctrl + Space 하면 System.out.println(); 으로 바뀐다.
  2. try 입력한 후 Ctrl + Space 하면 try-catch 문이 완성된다.
  3. for 입력한 후 Ctrl + Space 하면 여러가지 for 문을 완성할 수 있다.
  4. 템플릿을 수정하거나 추가하려면 환경설정/자바/편집기/템플리트 에서 할 수 있다.

메소드 쉽게 생성하기

클래스의 멤버를 일단 먼저 생성한다.

  1. override 메소드를 구현하려면 : 소스->메소드대체/구현 에서 해당 메소드를 체크한다.
  2. 기타 클래스의 멤버가 클래스의 오브젝트라면 : 소스->위임메소드 생성에서 메소드를 선택한다.

Organize import

  1. 자바파일을 여러개 선택한 후 소스 -> 가져오기 체계화 해주면 모두 적용된다.

소스 코드 형식 및 공통 주석 설정

  1. 환경설정 -> 자바 -> 코드 스타일 -> 코드 포멧터 -> 가져오기 -> 프로파일.xml 을 불러다가 쓰면 된다.

  2. 또한 다수의 자바파일에 프로파일을 적용하려면 패키지 탐색기에서 패키지를 선택한 후 소스 -> 형식화를 선택하면 된다.

  3. 환경설정 -> 자바 -> 코드 스타일 -> 코드 템플리트 -> 가져오기 -> 템플리트.xml 을 불러다가 쓰면 된다.
    에디터 변환

  4. 에디터가 여러 파일을 열어서 작업중일때 Ctrl + F6 키를 누르면 여러파일명이 나오고 F6키를 계속 누르면 아래로

  5. Ctrl + Shift + F6 키를 누르면 위로 커서가 움직인다.

  6. Ctrl + F7 : 뷰간 전환

  7. Ctrl + F8 : 퍼스펙티브간 전환

  8. F12 : 에디터로 포커스 위치

디버깅 단축키

  1. CTRL + Shift + B : 현 커서의 위치에 브레이크 포인터 설정/해제
  2. F11 : 디버깅 시작
  3. F8 : 디버깅 계속
  4. F6 : 한줄씩 실행(Step Over)
  5. F5 : 한줄씩 실행하되 함수일 경우 그 함수 내부로 들어감(Step Into)
  6. CTRL + R : 현재 라인까지 실행(Run to Line)
